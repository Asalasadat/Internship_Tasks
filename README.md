# AI/ML Internship - Week 1

## Day 1 — Environment Setup & Jupyter Workflow

In this day, I learned how to set up a reproducible Python environment for AI/ML projects using virtual environments, pip, and requirements.txt.  
I also practiced working with Jupyter Notebooks, including using code and Markdown cells to create clear and documented workflows.

Topics covered:
- Python virtual environments (venv) and package management using pip
- Creating and managing requirements.txt
- Jupyter Notebook workflow (Code & Markdown cells)
- Introduction to VS Code, Git, GitHub, and Google Colab
- Initializing a Git repository and committing project files

**Tools:** Python, venv, pip, Jupyter Notebook, VS Code, Git & GitHub


## Day 2 — Python for Data Science

In this day, I strengthened my Python fundamentals required for Data Science and AI/ML development.  
I practiced using Python data types, control flow, functions, list comprehensions, and basic Object-Oriented Programming concepts.

Topics covered:
- Python core data types (list, tuple, dictionary, set, etc.)
- Conditional statements and loops
- Writing reusable functions with parameters and return values
- List comprehensions for cleaner Python code
- Classes, attributes, and methods (OOP basics)

**Tools:** Python, Jupyter Notebook


## Day 3 — NumPy Fundamentals

In this day, I learned the fundamentals of NumPy, the core numerical computing library used throughout the AI/ML ecosystem. I practiced creating and manipulating arrays, performing vectorized operations, using boolean masking, and applying broadcasting for efficient numerical computations.

Topics covered:
- Introduction to NumPy and the ndarray data structure
- Creating arrays using `array`, `zeros`, `ones`, `arange`, `linspace`, and random functions
- Array attributes (`shape`, `dtype`, `ndim`)
- Indexing and slicing in 1D and 2D arrays
- Vectorized operations and boolean masking
- Broadcasting for operations on arrays with compatible shapes

**Tools:** NumPy, Jupyter Notebook
